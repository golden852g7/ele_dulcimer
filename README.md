# 电子扬琴微信小程序（基础版）

这是一个微信小程序（非 App）示例，包含两个核心能力：

1. **扬琴音阶模拟弹奏**：点击音阶按钮播放对应采样音。
2. **扬琴琴谱 CRUD**：支持琴谱新增、查看、编辑、删除（本地缓存）。

## 目录结构

- `pages/index`：电子扬琴演奏页面
- `pages/score`：琴谱管理页面
- `assets/audio`：扬琴标准音频采样目录

## 如何使用

1. 在微信开发者工具中导入本项目。
2. 将标准扬琴音频按命名规则放入 `assets/audio`（见 `assets/audio/README.md`）。
3. 运行后：
   - 在 **演奏** 页点击音阶按钮即可弹奏。
   - 在 **琴谱** 页可进行增删改查。

## 说明

- 琴谱数据使用 `wx.setStorageSync` 本地存储，便于快速验证功能。
- 若后续需要多设备同步，可把琴谱 CRUD 对接云开发/后端接口。
